About the Project

The Sign Language Recognition project aims to provide a tool that can help bridge the communication gap for the hearing and speech impaired by utilizing cutting-edge machine learning and computer vision technologies. This project is built using TensorFlow, Keras, and MediaPipe to recognize sign language gestures from videos or real-time webcam streams.

Our Technology Stack

  • TensorFlow for machine learning model development
  • Keras for building deep learning models
  • MediaPipe for hand landmark detection and preprocessing
  • Flask for the backend server
  • HTML, CSS, and JavaScript for the frontend interface

Our Team

We are a team of passionate developers and researchers dedicated to making sign language recognition technology more accessible. Our project was initiated in 2024, and we aim to continue improving its accuracy and expanding its functionality.
